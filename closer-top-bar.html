🎯 Цель виджета шапки

«Где я нахожусь по дисциплине + куда мне идти прямо сейчас» — это компас, не детальный отчёт. AI-разборы конкретных звонков, история переписки, статистика — всё это в виджете карточки сделки (правый sidebar). Виджет шапки про навигацию, не про разбор.

📋 Что в развёрнутом виджете (для клоузера)

  • 🎯 Чтобы выйти в следующую зону — конкретный план: «разгреби 1 из 8 → 7/35 = 20% → 🟡». Не общее «работай лучше», а математическое условие.
  • 📋 На сегодня — сколько 🔴 / 🟡 / 🟢 в работе. Сразу понятно что критично.
  • 🔥 Топ-3 по сумме — конкретные сделки которые нужно разгрести в первую очередь (большие деньги среди красных). Клик → переход в карточку сделки.
  • ⚡ Кнопка «Открыть дашборд» — если нужны детали по всем сделкам, статистика, история.

🎨 Цвет виджета меняется по зоне

  • 🟢 Зелёный — клоузер в норме (+30% к потоку лидов)
  • 🟡 Жёлтый — предупреждение (норма + сигнал)
  • 🟠 Оранжевый — поток лидов −50%
  • 🔴 Красный — 0 новых лидов · 🚫 блок

Попробуй переключить зону вверху страницы — увидишь как меняется виджет.

🧩 Разделение по виджетам

📌 Виджет шапки (этот)
Компас: где я, куда идти.
Зона + Топ-3 сделок + кнопка дашборда.

📲 Виджет карточки сделки
Разбор конкретной сделки: AI-цитаты звонка, что делать, история касаний, шкурный интерес, кнопки действий.

🛠 Технически (для программиста)

  • Точка монтирования: top_panel в amoCRM Widgets API · глобальная шапка
  • API: GET /api/widget/closer/{closer_id} · обновление polling 2 минуты
  • Реакция на события: webhook от amoCRM (звонок завершён, сделка изменилась) → форс-обновление
  • Авторизация: amoCRM OAuth token клоузера → API возвращает только его данные
  • Размер свёрнутой: 36-40 px высоты, не мешает работе
  • Подключение: single-page JS через manifest.json в amoCRM Marketplace

🎬 Интерактив на этой странице

  • Клик на полоску (или кнопку «▼ Раскрыть» сверху) → разворачивается виджет с 4 колонками
  • Клик на сделку в «Топ-3 по сумме» → симуляция перехода в карточку сделки (там виджет с AI-разбором)
  • Клик на кнопку «📊 Открыть дашборд» → ведёт на rop-workplace.html (полная аналитика)
  • Кнопки «🟢 / 🟡 / 🟠 / 🔴» вверху страницы → меняют зону клоузера (всё в виджете подстраивается)
